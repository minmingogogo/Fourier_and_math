# 从正态分布问题到傅立叶变换的联想

数学问题大多都彼此相似：
    一个预测任务：一堆变量X ，一个目标Y
### 怎么解呢？最简单方案：
回归的Y，线性回归；分类的Y，逻辑回归
### 有什么共同点？
都有变量的线性组合
### 问题1 为什么是线性组合？

假如解一个线性回归问题，自带两前提
1 变量独立同分布 2 误差服从正态分布
然后咔咔咔开始一顿魔幻的相关性检验，交叉特征构建
### 问题2 为什么假设独立同分布，误差为什么正态？

实际回归预测中大部分模型都是广义线性模型，都离不开指数分布族，为什么呢？
因为指数分布族性质好？
公式优美，参数又少，还共轭，先验推后验无压力。
